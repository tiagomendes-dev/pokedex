## Visão Geral

Este é um projeto de Pokédex desenvolvido em JavaScript, que permite listar e visualizar informações detalhadas dos 150 primeiros pokémons da série original. A ideia por trás deste projeto é fornecer uma interface simples e amigável para os fãs de Pokémon explorarem suas criaturas favoritas.

## Instalação

1. Clone o repositório para sua máquina local.
2. ```git clone https://github.com/tiago-mends/pokedex.git```
3. Abra o arquivo app.js em seu navegador.
4. Navegue pela lista de pokémons e conheça-os.

## Funcionalidades

- Lista paginada dos 150 primeiros pokémons, exibindo números na Pokédex.
- Detalhes de cada pokémon, incluindo tipo e nome.
- Layout responsivo para melhor experiência em dispositivos móveis.
